#1 OBjetivos

1- O que são estruturas condicionais?;
2- Estruturas de decisão;
3- Estruturas de repetição;


- Estruturas condicionais são instruções para realizar determinadas tarefas a partir de uma condição, seja de decisão ou repetição;
- Ex: Um jogo precisa mudar o placar toda vez que um jogar marca uma pontuação;



// Estruturas de decisão 

if 

Podemos usar as palavras reservadas "if" para estabelecer uma condição:
Ex : var jogador1 = 0;
        var jogador2 = 0;
        if(jogador1 > 0) {
          console.log("jogador1 marcou ponto");
        }

        // Como lçer: se o jogador1 tiver valor maior que 0 ele marcou ponto;


  # Laços  de repetição 
  - São estruturas condicionais que repetem uma instrução até atingir determinada condição;

  - for;
  - for/in;
  - for/of;
  - while;
  - do/while;
